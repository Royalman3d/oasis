
# Oasis

A cinematic, single-page web experience exploring deep-sea species. This project focuses on high-end transitions, minimalist UI, and smooth performance across mobile and desktop.

## Overview

Oasis is a "Film-Strip" style website. Instead of traditional scrolling, it uses a controlled panel system where each section feels like a different scene in a movie. It specifically showcases deep-sea creatures (Oceania) using AI-generated video and smooth motion design.

### Key Technical Details

- Mobile-Ready: Optimized with dvh units and Safe-Area insets to look perfect on iPhones and Androids.

## Tech Stack

- Core: React + TypeScript

- Styling: Tailwind CSS

- Animations: Framer Motion

- Visuals: AI-generated video assets

## How to Run
